# Code of Conduct

Seahorse is an open research software project. We want discussions, issues, and
pull requests to stay constructive, technical, and welcoming to contributors at
different experience levels.

## Expected Behavior

- Be respectful and professional in public discussions.
- Focus critique on code, documentation, experiments, and reproducibility.
- Give actionable feedback when requesting changes.
- Respect maintainers' time by keeping issues and pull requests scoped.
- Credit prior work and related implementations where relevant.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Insulting or dismissive comments about contributors or users.
- Publishing private information without explicit permission.
- Repeated off-topic posting or disruption of technical discussions.
- Plagiarism or misrepresentation of research, code, or results.

## Reporting

If you see behavior that violates this code of conduct, contact the repository
maintainers through GitHub. If the report includes sensitive information, open a
minimal non-sensitive issue asking for a private contact channel rather than
posting private details publicly.

Maintainers may remove comments, close discussions, block users, or take other
moderation steps needed to keep the project healthy.

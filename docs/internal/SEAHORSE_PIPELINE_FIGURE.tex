\caption{%
    \textbf{Seahorse benchmarking pipeline.}
    Blue-shaded boxes constitute the shared, frozen benchmark contract applied
    identically to all presets.
    Orange-shaded boxes are family-owned: each \textsc{PresetDescriptor}
    derives data-dependent quantities (coordinate transforms, spatial support
    bounds) from the training partition only, injecting them before model
    instantiation.
    All four execution modes (\texttt{fit}, \texttt{tune}, \texttt{bench},
    \texttt{evaluate}) operate under the same frozen policy.
    The \textsc{RunResult} artifact records first-class reporting metadata
    that distinguishes training objective, NLL kind and reporting space,
    evaluation exposure, and reproducibility tier.%
}
